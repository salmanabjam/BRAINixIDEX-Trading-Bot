## 💼 مدیریت ریسک

### تنظیمات پیش‌فرض:
```python
سرمایه اولیه: $10,000
ریسک هر معامله: 1.5%
حداکثر موقعیت: 20%
نسبت سود/ضرر: 2:1
```

### استراتژی Stop Loss:
```
Stop Loss = قیمت ورود ± (ATR × 2)
Take Profit = قیمت ورود ± (ATR × 4)
```
